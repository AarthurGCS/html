O comando pasta/documents/estudos html.com.ww está sendo envelopado com a tag CODE, que melhora a visualização desse tipo de escrita.

Tambem podemos utilizar a tag PRE para representar um código em JavaScript na forma que está sendo escrita no html. Abaixo está um exemplo: 

Lorem ipsum dolor sit amet consectetur 

adipisicing elit. Inventore autem magnam 

quia modi aliquid, nam quis eveniet 

tempore mollitia, quod distinctio in est 

id natus unde velit, quae nihil amet.
